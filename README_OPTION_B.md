# xv6 Priority Scheduler - Application Track (Option B)

This section contains the deliverables for **Option B: Application Track - Implementation Evaluation & Enhancement Proposal**.

## Files Included

1.  **[technical_report.md](technical_report.md)**: A comprehensive evaluation of the Priority Scheduler feature.
    -   **Evaluation**: Analysis of system behavior under normal and stress loads.
    -   **Limitation**: Identification of process starvation as a critical inefficiency.
    -   **Enhancement Proposal**: Design and pseudocode for implementing **Process Aging** to solve starvation without modifying kernel code.
    -   **Analysis**: Trade-offs between performance, stability, and complexity.
